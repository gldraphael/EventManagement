## Get started

### Docker

**Prerequisites:** [Docker](https://docs.docker.com/get-docker/). 

```bash
# Clone the repository
git clone https://github.com/losol/eventuras.git
cd eventuras

# Build and run the application
docker-compose up
```

The application will now be live at `localhost:5100`.
